# 原著主线对齐规则

## 核心原则
1. **时间线**：玩家年级 = 哈利·波特同期学年（同年级事件同步发生）。
2. **哈利仍是核心**：魔法石、密室、伏地魔复活、邓布利多之死等关键节点**必须发生**；玩家是目击者、参与者或旁观者，不是替代哈利的天选之女。
3. **个人线自由**：恋爱、社团、学院分、日常摩擦可自由展开，但不得打断或取消原著主线。
4. **每 8–12 回合**须推进或明确呼应 `eventContext.canonPlot.dueBeat` 或 `nextMandatory`。
5. 到达 beat 对应周次时，**必须**在 narrative 中安排相关场景、背景议论或 aftermath。

## 玩家角色定位
- 可与哈利、赫敏、罗恩同场（魁地奇、课堂、万圣节、D.A. 等）
- 可在主线事件中帮忙、受伤、见证、提供情感支持
- **不可**：独自击败伏地魔、代替哈利完成预言、改变「伏地魔已复活」等既定事实
- 四年级勾选「拯救塞德里克」时：塞德里克可存活，但伏地魔复活仍须发生

## 高年级开局（2–7 年级）
- 此前学年原著主线视为**已发生**（见 state.flags.canonPlot.completed 背景）
- 叙事可闪回或对话提及，但勿重复完整剧情；聚焦本学年主线

## 标记主线进度
剧情推进后更新：
```json
"flags": {
  "canonPlot": {
    "completed": ["troll_halloween", "first_quidditch"]
  },
  "eventsTriggered": ["troll_halloween"]
}
```
`completed` 使用 beat id；系统会自动加年级前缀。

## 与日常/感情线的平衡
- 主线 beat 周：选项中至少 1 个与主线相关（如「去魁地奇场看比赛」「打听巨怪传闻」）
- 非 beat 周：日常、恋爱、霍格莫德、图书馆均可
- 禁止连续 3 回合完全无视当前学年主线背景（即使在小场景也应有人议论）

## 禁止
- 编造与原著矛盾的世界级事件（如一年级就办三强赛）
- 让配角取代哈利的核心行动（如玩家独自找魔法石）
- 省略本学年 mandatory beat（除非玩家年级不对应该事件）
